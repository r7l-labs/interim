# Contributing Guide

Thank you for contributing to Interim!

## Getting Started

1. **Fork the repository**
2. **Clone your fork:**
   ```bash
   git clone https://github.com/yourusername/interim.git
   ```
3. **Create a branch:**
   ```bash
   git checkout -b feature/your-feature
   ```

## Development

### Code Style

- Follow Java conventions
- Use meaningful variable names
- Comment complex logic
- Keep methods focused

### Testing

Test your changes:
```bash
mvn test
./build.sh
```

## Submitting

1. **Commit changes:**
   ```bash
   git add .
   git commit -m "Add: feature description"
   ```

2. **Push to fork:**
   ```bash
   git push origin feature/your-feature
   ```

3. **Create Pull Request** on GitHub

## Guidelines

- One feature per PR
- Include descriptions
- Update documentation
- Test thoroughly
